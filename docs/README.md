## 🔍 查找文档

按日期查找：
```bash
ls docs/logs/*20260111*
```

按关键词查找：
```bash
grep -r "Agent" docs/ --include="*.md"
```

## ✏️ 命名规范

### 报告
```
{类型}_{主题}_{日期}.md
例: AI_Agent_重构报告_20260111.md
```

### 日志
```
{类型}_{日期}_{主题}.md
例: 开发日志_20260111_Agent重构.md
```

### 设计文档
```
{系统名}_{版本/特性}.md
例: 牛股信号分析系统_合规优化版.md
```
