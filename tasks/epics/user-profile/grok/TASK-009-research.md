# TASK-009: 问诊问题设计调研

## 背景 (Why)
在实现问诊引擎前，需要研究交易心理学领域的最佳实践，确保我们的问题设计科学有效。

## 目标 (Done Definition)
- [ ] 调研交易心理学的用户画像维度
- [ ] 对比现有交易平台的用户问诊流程
- [ ] 优化 5 个核心问题的措辞
- [ ] 建议 10 个扩展问题的设计
- [ ] 输出调研报告

## 调研方向

### 1. 交易心理学研究
- 交易者常见心理弱点分类
- 风险偏好测量的标准方法
- FOMO/损失厌恶的学术定义

### 2. 竞品分析
- 同花顺/雪球 的用户问卷设计
- 美股平台 (Robinhood, TD Ameritrade) 的风险评估问卷
- 量化交易平台的用户画像收集方式

### 3. 问题设计优化
- 每个问题应该避免引导性
- 选项设计应该覆盖完整光谱
- 中文措辞的自然度
